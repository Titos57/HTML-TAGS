Everything you need to remember!!

HTML is the base of all web pages. It is the first layer, where you include your text, images and videos. After learning all the basics for HTML, you can start adding effects and styling with CSS and JavaScript. Below you can find everything you need to know to start your own website.

HTML Basics

Tags Common Attributes Description
<!DOCTYPE html> Empty At the very start of the html file, declares the html version
<html> Empty Contains the whole html code, it is the starting declaration
<head> Empty Contains all the tags not shown to the user
<title> Empty Placed inside the head tag to set what name the users browser tab will have
<body> Empty Contains all the content shown to the user
Up to this point, it's all the necessary tags for a HTML code to work
<p> Empty Contains the paragraphs of a website
<h1-6> Empty Contains a header with 1 being the bigger one and 6 the smallest one
<strong> Empty Turns the text into bold
<em> Empty Turns the text into italic
<span> Empty Used inside text to separate a small area for styling
Semantic Knowledge, that the browser expect you to follow
<header> Empty Used to enclose headers in it
<nav> Empty Used to enclose our navigation links of the page in it
<div> Class="name of class" or id="name of id" Used to separate parts of the website in smaller groups
<section> Class, Id Similar to div tag, used to split content of the page to smaller groups(recommended for use)
<article> Empty Used to specify in our code where our content is(recommended to use)
<figure> Empty Used to enclose images, diagrams etc, of the page
<figcaption> Empty Used inside the figure tags to add a caption to the content added(images, diagrams etc)
<footer> Empty Used in the end of our page to enclose all the information about the page(contact info, emails, phone number etc)
Lists
<ol> Empty Declares an organized list with numbers
<ul> Empty Declares an unorganized list with bullet points
<li> Empty Used to add elements to both types of lists
Adding Content(links, images, videos, gif etc)
<a "attribute">"Text that the user clicks on to load link"</a> href="Url or relative path" or href=#id (id-name is placed somewhere in the site so whenever link is clicked it moves user to it), target="_blank"(used to open the link in another tab) Used to add links to the page
<img "attribute" /> src="relative path or url", alt="A description of the image" - (Needed for blind people) Adds an image to the site
<video "attribute">Video not Supported</video> src="relative path or url", height="height of the video", width="width of the video", autoplay - plays automatically when the user loads the page, controls - adds control buttons to the video, loop - for it to loop when ends Used to add video to a site
<audio "attribute"> <source src="path to audio file.mp3" type="audio/mp3"> </audio> autoplay - plays automatically when the user loads the page, controls - adds controls buttons to the audio Adds audio file to the site
<embed "attribute"/> src="gif file path" Adds a gif to the site

Note!: Almost every tag can take an id and class attribute, in case we want it to be customized with CSS later on.



Table Tags

Tags Common Attributes Description
<table> border="number" - Adds a border to the table The starter element that contains the whole table
<thead> Empty Contains the column headers of the table
<th> Empty Used inside the <thead> to add headers to the table
<tbody> Empty Contains the actual data of the table
<tr> Empty Used to create rows in the table
<td> colspan, rowspan ="number" - Defines how many columns or rows a data element will span Used to add data to a row
<tfoot> Empty Used in the end of the table to contain some total values or data results
